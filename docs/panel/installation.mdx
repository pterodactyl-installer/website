# Installation

Now we a ready to start the installation process.

## Running the script

First, we need to make sure that the `root` user is selected.

```bash
sudo -i
```

Then run the script itself.

```bash
bash <(curl -s https://pterodactyl-installer.se)
```

:::caution
Do not put `sudo` in front of the command above. It will not work.
:::

:::info
If you get `bash: curl: command not found`

Install the `curl` package
:::

## Selection

You will be greeted with a selection of what to install.
Choose option **0** to install the panel.
