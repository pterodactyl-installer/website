# Database

Setting up a database.

## Database name and username

First you will be prompted for a database name and a database user username.  
You can use any names you want, but it is best to keep the default ones.
:::tip
Pressing **ENTER** will select the default option.
:::

```
* Database name (panel):
* Database username (pterodactyl):
```

## Database password

Then it will ask for a database password.
:::caution
Pressing **ENTER** will generate a random password.

You can find the password after installation in `/var/www/pterodactyl/.env`
:::

```
* Password (press enter to use randomly generated password):
```
