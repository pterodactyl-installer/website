# Configuration

## Timezone

Then you will be asked for a timezone. This timezone will be used for servers and schedules.  
Here is a handy tool to find your timezone:

<TimeZoneSelect />

## Email

Then you will be asked for an email address.
This email address will be used for default egg author and letsencrypt.

```
* Provide the email address that will be used to configure Let's Encrypt and Pterodactyl:
```

## Intitial user

Then you will be asked for initial admin account details.

```
* Email address for the initial admin account:
* Username for the initial admin account:
* First name for the initial admin account:
* Last name for the initial admin account:
* Password for the initial admin account:
```
